Touch Target Verification Tool

WCAG 2.1 Level AAA Requirement

Success Criterion 2.5.5 Target Size (Level AAA):
The size of the target for pointer inputs is at least 44 by 44 CSS pixels.

This tool helps verify that all interactive elements on the Eternal Physios clinic website meet the minimum touch target size requirement.

How to Use This Tool

  1. Open the clinic website at http://localhost:3000
  2. Open DevTools (F12) and enable Device Toolbar (Ctrl+Shift+M)
  3. Set viewport to the sizes listed below
  4. Right-click each interactive element → Inspect
  5. Check the computed dimensions in the Styles panel
  6. Verify width × height ≥ 44px × 44px

Manual Verification Checklist

Use DevTools to manually verify these measurements on the live site:

  1. Open http://localhost:3000
  2. Open DevTools (F12)
  3. Enable Device Toolbar (Ctrl+Shift+M or Cmd+Shift+M)
  4. For each viewport size (375px, 414px, 768px, 1024px, 1280px):
    • Set the viewport width
    • Scroll down to trigger CTABar
    • Right-click CTABar Call button → Inspect
    • Check computed width and height in Styles panel
    • Verify width × height ≥ 44px × 44px
    • Repeat for WhatsApp button
    • Click through service cards and verify clickable area
    • Click through doctor cards and verify clickable area
    • Test all other interactive elements
